### 📂 Featured Projects

*   **[Honey Shop](https://github.com/Lukasz-Sochacki/honeyShop)** - A comprehensive full-stack artisanal e-commerce application. Features a dynamic variant system, persistent shopping cart, and secure checkout form validation. Built with *React, Redux, NestJS, Prisma, and PostgreSQL*.

*   **[MusicApp](https://github.com/Lukasz-Sochacki/MusicApp)** - A modular music streaming web application built using Object-Oriented JavaScript (OOP). It integrates AJAX/Fetch API to dynamically consume song metadata and playlists from a REST database. Features an elegant custom single-page client-side router using URL hashes, category-based filtering, advanced song discovery algorithms, and a seamless global HTML5 custom audio player interface.

*   **[Pizzeria](https://github.com/Lukasz-Sochacki/JavaScript---pizzeria)** - A complex interactive restaurant ordering system designed using modern ES6 modular JavaScript and semantic templates. Features an advanced shopping cart engine with multi-item data synchronization, dynamic price calculation structures, and an encapsulated interactive booking system/widget to manage table reservations dynamically.
